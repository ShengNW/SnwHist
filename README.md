# SnwHist

这是一个供 Codex 之间交接与检索的统一接口仓库。按下面步骤即可开始使用。

## 使用步骤

1. 克隆仓库到本机（通常放在 `$HOME/` 下）。
2. 复制下面这段提示词到需要继承背景信息的 Codex 会话中（按实际存放目录修改）：

```
对话历史存放在$HOME/SnwHist中，请阅读index.md找到与本次对话相关的交接文件
```

3. 需要新增或更新根目录关键 md 时，同时更新 `AGENTS.md` 索引说明。
4. 需要加入子目录时：
   - 将子目录作为独立 Git 仓库（子模块）添加。
   - 在 `index.md` 登记其元信息。

## 约束提示

- 不编辑子文件夹中的 md 文档。
- 子目录维护者更新子目录后，需要先提交到子目录仓库，再提交并 PR 更新 `index.md`。

更多规则请见 `Interface.md`。
